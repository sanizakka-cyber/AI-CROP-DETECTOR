## Image Specifications & Requirements

### File Format & Optimization
- **Format:** JPEG (.jpg) or WebP (.webp) - both supported
- **Compression:** Optimized for web (quality ≥ 75%, file size <500KB each)
- **Resolution:** Minimum 800×400px for service cards; 400×400px for gallery
- **Encoding:** Progressive JPEG recommended for faster perceived load

### Accessibility Requirements
- **Alt Text:** Descriptive, specific to content (not "image" or generic)
- **Provided:** ✅ All alt texts included in manifest above
- **Implementation:** Already in code via `alt={description}` attributes

### Responsiveness
- **Mobile:** 100% container width, lazy-loaded
- **Tablet:** Responsive grid (2-3 columns gallery)
- **Desktop:** Full resolution
- **Implementation:** ✅ Next.js `Image` component handles all breakpoints

### Branding Requirements
- **Color Palette:** Green/emerald (emerald-700 borders, teal accents)
- **Consistency:** Professional, cohesive aesthetic
- **Quality:** High-quality, well-lit photography
- **Content:** Authentic farm/agricultural operations (Northern Nigeria preferred)

### Licensing & Rights
- **Requirement:** Commercial use rights confirmed
- **Source Options:**
  - Original photography from your farms ✅ (Recommended - most authentic)
  - Licensed stock photos (Unsplash, Pexels, Pixabay, Shutterstock)
  - Professional photographer commission
- **Attribution:** Not required but appreciated for third-party content
